== Installation ==
Instalace je velmi jednoduchá

1. Jděte do pluginy - Přidat nový plugin
2. Vyhledejte WooCommerce čeština
3. Dejte instalovat
4. Klikněte na aktivovat plugin
5. Klikněte na nastavení - WooCommerce translate a aktivujte
5. Hotovo

Instalace a ukázka:

[youtube https://www.youtube.com/watch?v=HNeMGQ89bjA]
